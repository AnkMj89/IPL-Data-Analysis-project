# IPL-Data-Analysis-project
# IPL Data Analytics Challenge 2026

## Overview

This project analyzes IPL cricket data using Python, Pandas, and Matplotlib. The project explores team performances, player statistics, toss impact, and scoring patterns using IPL datasets.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Google Colab

## Dataset Used

* matches.csv
* deliveries.csv

## Analysis Performed

1. Teams with most IPL wins
2. Top run scorers
3. Top wicket takers
4. Toss impact analysis
5. Powerplay vs death over analysis

## Key Insights

* Some IPL teams consistently dominate the tournament.
* Top batsmen contribute significantly across seasons.
* Death overs produce aggressive scoring patterns.
* Toss advantage exists but does not guarantee victory.

## How To Run

1. Open the notebook in Google Colab
2. Upload matches.csv and deliveries.csv
3. Run all notebook cells
